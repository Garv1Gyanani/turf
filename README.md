# Sports Villa — Multi-Sport Arena

A premium, modern multi-sport facility booking and academy landing page for **Sports Villa**, located in Nirman Nagar, Jaipur.

## Features
- **Seamless WhatsApp Booking**: Users can choose their sport, date, and time slot to book instantly via a glassmorphic pop-up modal.
- **Fully Responsive & Fluid**: Sleek layout engineered for maximum accessibility on mobile, tablet, and desktop screens.
- **Training Academies**: Dedicated enrollment section for structured training programs (Cricket, Football, Badminton, and Youth Development).
- **Interactive Tournaments**: Live schedule updates for upcoming leagues and brackets.
- **High-Quality Gallery**: Showcase of premium floodlit turfs and indoor courts.

## Tech Stack
- **Frontend**: React, Vite
- **Styling**: Vanilla CSS (Modern custom variables, fluid grid system, and glassmorphic overlays)
- **Icons**: SVG vectors
